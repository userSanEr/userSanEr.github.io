uniapp2wxpack

发表于 2021-11-11 分类于 uniapp 阅读次数：

这是uniapp插件市场提供的一款可以实现uniapp框架与原生小程序汇写以及解耦的一个插件，在文档主要介绍了插件的使用说明。

1、使用流程

  • 使用vue cli创建uni-app项目，在创建项目过程中选择模板时，默认回车即可

搭建步骤

  • 首先要具备node/vue-cli3开发环境

  • 创建项目 vue create -p dcloudio/uni-preset-vue my-project

  • 启动项目 cd my-project进入项目目录 npm run dev:mp-weixin

    注：

    1、为什么没有直接用npm server？

    因为我们需要小程序编译所以使用这中方式，在项目搭建之后的 package.json中我们都可以看到

    2、如果vue-cli版本为2.0版本可输入一下命令

    npm uninstall -g vue-cli卸载2.0版本脚手架

    npm install -g @vue/cli安装2.0版本

    3、在微信小程序中进行编译要注意不能将整个文件夹放进去，这样小程序不会识别

    我们需要找到my-project/dist/dev/mp-weixin将这个文件放入开发者工具，之后微信小程序便会自动监听更改，进行编译

  • 安装uniapp2wxpack插件 npx uniapp2wxpack --create

  • 运行开发命令npm run dev:mp-weixin-pack

  • 使用小程序开发者工具预览dist/dev/mp-weixin-pack目录

3、快速上手

页面底部的tabbar统一在原生app.json中进行配置

uniapp中的原生页面统一放在wxresource中，包括js/css一切需要使用原生语法写的内容

如果要在wxResource配置的页面中访问app.vue中的某个变量

在dist/dev/mp-weixin-pack中存在的内容原生以及uniapp混合编译之后的目录，与原生小程序目录整体相同

在dist/dev/mp-weixin中主要是uniapp编译之后的目录

在这里就dist/dev/mp-weixin-pack目录进行示例

在进行学习时，可以参考dist/dev/mp-weixin-pack打包之后的目录进行对比学习，更有助于多方面的学习该组件。

4、uniapp中设置分包

  • 对原vue文件进行分包

  • 对使用原生语法的文件进行分包设置

5、编译之后的内容

在uniapp中若使用vuex进行全局存储，则在编译之后，使用vuex的该页面的js中的大量编译内容中，就会出现你使用的vuex的模块内容，uniapp原理应是在js中对所有使用vue编译的 内容转换为原生小程序语法。

在uniapp设置的分包编译之后，所有的页面仍编译到uniSubpackage的pages文件夹下

在使用vuex时与原来的用法相同，没有任何区别
